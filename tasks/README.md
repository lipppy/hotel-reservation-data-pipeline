# Tasks

Work through these files in order: `01_task.py`, `02_task.py`, ... Each one
has a docstring explaining what it does and `TODO` comments marking what you
need to write.

See [PROJECT.md](../PROJECT.md) for the full flow these tasks build up to,
and a suggested day-by-day schedule.

Run any task with:

```bash
make run FILE=tasks/01_task.py
```

There is no `06_task.py`. The final step is done directly in
[`api.py`](../api.py): combine everything from tasks 1-5 into the real
`/api/reservations` endpoint.
